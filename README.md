# podzspace-backend

Backend for the Podcast Studio Marketplace (US-focused).

This repository is intentionally minimal so you can create a GitHub Codespace. Application code (Express + MongoDB + Stripe Connect) will be added in subsequent commits.

## Planned Stack
- Node.js, Express, Mongoose, Luxon
- Stripe Connect (Destination Charges)
- REST endpoints for studios, rooms, availability, bookings
